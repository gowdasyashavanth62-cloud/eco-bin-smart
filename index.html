Smart Waste Management System

Alva's Institute of Engineering & Technology, Moodbidri

🎯 Purpose

Real-time smart waste monitoring and waste collection optimization using IoT sensors and intelligent data analytics for a cleaner, greener campus environment.

🔧 Hardware Components

📡
Ultrasonic Sensor (HC-SR04)

Real-time fill-level detection and distance measurement inside smart bins.

💧
Moisture Sensor

Capacitive moisture sensor for wet/dry waste classification and segregation.

📶
NodeMCU (ESP8266)

Wi-Fi enabled microcontroller for data transmission and cloud connectivity.

🏗️ System Architecture

Sensors→NodeMCU→Wi-Fi→Cloud→App
